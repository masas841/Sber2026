# СберКопилка vs монстры-расходов

Автономная аркада для **одной машины** на фестивале: Pac-Man на 60 секунд, таблица лидеров за день, управление **USB-джойстиком**.

## Геймплей (по ТЗ)

| Элемент | Очки / эффект |
|---------|----------------|
| Монета `.` / `o` | +10 |
| Процент `%` | +25 |
| Лого (большая точка) | +50 |
| **Портфель** (звезда в центре) | 8 с: неуязвимость, **x2** очки, можно «съесть» **Инфляцию** (+200) |
| Враги | **Импульс** (хаотичный), **Инфляция** (преследует на дистанции), **FOMO** (быстро преследует) |
| Жизни | 3 |
| Раунд | 60 с (настраивается в `.env`) |

## Управление — джойстик

Используется **Gamepad API** (стандарт для USB arcade stick в Chrome/Edge):

| Действие | Джойстик |
|----------|----------|
| Движение | Стик (оси 0/1) или D-pad (кнопки 12–15) |
| Старт / подтверждение | Кнопка **0** (A), **9** (Start) или **7** |
| Смена имени после игры | Влево / вправо на стике |

Внизу экрана индикатор:

| Текст | Значение |
|-------|----------|
| Зелёный: `DualSense Wireless…` | Геймпад активен, можно играть |
| Жёлтый: `… — нажмите любую кнопку` | Устройство видно, нужно «разбудить» Chrome |
| Красный: `нажмите любую кнопку…` | Пока нет нажатия (норма для DualSense) |

### DualSense (PS5) в Chrome

1. Откройте **http://127.0.0.1:8766** и **кликните по странице** (фокус вкладки).
2. **Нажмите любую кнопку** на геймпаде (Cross, Options, D-pad) — без этого Chrome не заполняет `getGamepads()`.
3. Старт игры: **Cross (×)** или **Options**.
4. Движение: **левый стик** или **D-pad**.

Bluetooth и USB работают одинаково через Gamepad API.

**Для теста без геймпада:** стрелки на клавиатуре, Enter/Space = Start.

Мёртвая зона стика: `JOYSTICK_DEADZONE=0.35` в `.env`.

### Настройка Ultimarc / Xin-Mo / «как клавиатура»

Если стик прошит как **HID Gamepad** — работает сразу. Если только клавиши WASD — в README gigavibe-стиля можно добавить отдельный драйвер; для аркады рекомендуется режим **gamepad**, не keyboard.

## Запуск

```powershell
cd i:\Cursor\Sber2026\sberkopilka
.\run.ps1
```

Откройте **http://127.0.0.1:8766** в Chrome полноэкранно (`F11` или `--kiosk`).

Проверка API: `http://127.0.0.1:8766/api/health`

## Лидерборд

- Топ за **календарный день** (локальное время ПК)
