# ИнвестКопилка — дизайн (из PDF)

**Источник:** [ИнвестКопилка.pdf](ИнвестКопилка.pdf)  
**Разрешение LED-экрана:** **504 × 672 px** (портрет 3:4)

## Экраны (поток)

1. **Старт** — логотип «ИнвестКопилка / монстров-расходов / против», кнопка «играть», лёгкая анимация фона (трава, копилка, цветы, 2D-элементы).
2. **Онбординг** (3 карточки, автолистание) — «Собирай капитал», «Уворачивайся от препятствий», «Ворвись в топ»; переходы с прокруткой шара.
3. **Игра** — лабиринт, счёт в облаке внизу (`счёт: 000`), жизни (сердца; при потере — голубое).
4. **Результат** — счёт, диссолв поля; варианты: не в рейтинге / в рейтинге / рекорд (конфетти).
5. **Таблица дня** — «Ворвись в топ», до 7 строк + текущий гость; логотип фиксирован сверху.
6. **Простой** — «инвестКопилка ребалансирует портфель…».

## Игровое поле

| Объект | Смысл |
|--------|--------|
| Монеты | Баллы |
| % | Проценты |
| Фонды, регулярные пополнения, акции | Активы СберИнвестиций |
| Импульсивные покупки, FOMO, инфляция | Враги («злодеи») |
| Портфель СберИнвестиций | Бонус: неуязвимость + усиление |

## Реализация в коде (текущее)

| Параметр | Значение |
|----------|----------|
| Canvas Phaser | `504 × 672` (`sberkopilka/web/js/maze.js`) |
| Лабиринт | 19×15 клеток, `TILE = 26` |
| Масштаб на мониторе | `Phaser.Scale.FIT` + CSS 3:4 |

## Анимации (гибрид iframe + Phaser)

| Эффект | Реализация |
|--------|------------|
| Шевеление декора (трава, копилка, звезда) | CSS в `figma-animations.css` → живой JSX в iframe |
| Переходы онбординга (прокрутка шара) | `postMessage` → классы `kop-transition-in/out` |
| Автолистание онбординга | таймер 6 с в `FigmaFlowScene` |
| Диссолв игра → результат | `cameras.main.fadeOut` в `GameScene` |
| Конфетти на рекорде | DOM-частицы в `figma-animations.js` |

PNG в `assets/figma/screens/` — только fallback без CDN.
